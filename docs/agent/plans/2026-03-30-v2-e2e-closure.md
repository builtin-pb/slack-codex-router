# Slack Codex Router v2 E2E Closure Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Close the remaining `v2` runtime gaps so Slack threads run in per-thread worktrees, App Server events render back into Slack, Slack controls execute real actions, and the root entrypoint no longer depends on `legacy/v1` assumptions.

**Architecture:** Keep `RouterService` responsible for project lookup, thread routing, and durable thread metadata; keep `router/runtime.ts` responsible for App Server event bridging and recovery; extend `slack/app.ts` into a real Slack interaction surface by adding action handlers that call back into `RouterService` and runtime-owned helpers. Avoid large refactors and do not route production behavior through `legacy/v1`.

**Tech Stack:** TypeScript, Node.js, Slack Bolt, Codex App Server over stdio, SQLite via `better-sqlite3`, Vitest

```ts
export async function requestRouterRestart(input: {
  store: RouterStore;
  slackChannelId: string;
  slackThreadTs: string;
  requestedAt?: string;
}): Promise<{ exitCode: number }> {
  input.store.recordRestartIntent({
    slackChannelId: input.slackChannelId,
    slackThreadTs: input.slackThreadTs,
    requestedAt: input.requestedAt ?? new Date().toISOString(),
  });

  return { exitCode: RESTART_EXIT_CODE };
}
```

```ts
export async function getRepositoryStatus(input: {
  repoPath: string;
  branchName: string;
  targetBranch: string;
  run(input: { args: string[]; cwd: string }): Promise<{ stdout: string }>;
}): Promise<{ repositoryName: string; sourceBranch: string; targetBranch: string; worktreeStatus: string; checksStatus: string }> {
  // git status --porcelain -> clean/dirty
  // derive repo display name from path basename
  return {
    repositoryName: basename(input.repoPath),
    sourceBranch: input.branchName,
    targetBranch: input.targetBranch,
    worktreeStatus: "clean",
    checksStatus: "not run",
  };
}
```
